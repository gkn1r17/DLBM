Environmental Selection

Changing the selection function

Selection-related settings live in SciConfig, including W, TEMP_FILE, TEMP_START_RANGE, and TEMP_MUTINTV. If you want to alter how fitness depends on environment, start by tracing the selective lineage and grid-box logic that uses lineage temperature optima.

Adding additional selection functions

A practical approach is to introduce a new strategy or branch alongside the existing selective logic rather than modifying every call site at once. Keep the configuration switch explicit so runs remain reproducible.

Dispersal

Dispersal behaviour is centered in the dispersal package and tied to the transport matrix representation. Changes here can affect both local correctness and distributed communication, so small validation runs are recommended first.

Mutation

Mutation rate is configured through MUTATION. Mutation-related output and debug checks are coordinated through inputOutput.Output and the phylogeny utilities. If you change mutation logic, also review mutant save timing and debug tooling.

Growth / Mortality

Growth and mortality are derived from daily rates and converted internally to timestep-scale values in Config. Any changes should preserve consistency between the configured daily rates and the timestep sizes.

Initialisation

Initial populations are either loaded from previous output or created from the configured initial state. The setup flow in Runner shows where new initialisation pathways can be inserted safely.

Additional Dimensions of Diversity

To add extra tracked traits such as size class or functional group, plan the change across lineage storage, mutation rules, save/load formats, and comparison tests. This type of extension usually touches both in-memory structures and file formats.

Output and Metrics

Output currently includes full state saves, abridged time series, checkpoint files, and optional mutant/phylogeny information. New metrics are often easiest to add first to the reporting CSV path and then, if useful, to full save output.
