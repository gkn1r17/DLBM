Tracer Mode

Tracer mode is intended for special-purpose runs rather than standard production simulations. It changes the assumptions behind the usual ecological dynamics, so outputs should be interpreted separately from normal runs.

Resting Spores

Dormancy-related behaviour, including size-refuge style logic, should be treated as experimental unless it has been validated for your exact scenario. If you rely on it for published or production work, add explicit scenario tests and inspect the output carefully.
