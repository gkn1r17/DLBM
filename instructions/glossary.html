Key Terms

Lineage
A tracked population identity represented in the lineage classes. Lineages are the main units of diversity in the model.
Neutral lineage
A lineage without environmental trait tracking beyond abundance and identity.
Selective lineage
A lineage with an environmental optimum, such as temperature preference, allowing environmental selection to act.
Grid box
A spatial location in the transport matrix. Each grid box stores local populations and receives dispersal from connected locations.
Transport matrix (TM)
The connectivity data describing movement among grid boxes. DLBM can load this from files or generate an artificial matrix for simplified cases.
Dispersal timestep
The interval, in hours, between transport events. This is controlled by DISP_HOURS.
Growth timestep
The interval, in hours, between growth and mortality updates. This is controlled by GROWTH_HOURS.
Checkpoint
An intermediate save file used to restart long runs when cluster time is limited.
Phylogeny output
Saved mutant history produced when mutation tracking is enabled and mutant save times are configured.
Controller node
The node responsible for central logging and some output coordination when running in distributed mode.
